Typesetting software

  • Installer for the MathTime Professional 2 fonts on Tex Live in Linux or macOS. I highly recommend you move away from using the monotonous Computer Modern Latex fonts and use MathTime instead. It's free!
  • The Student Handouts Latex package provides an easy to use mechanism for managing the production of handouts to be used in class. It also provides some changeable handout styling, and comes with pretty extensive documentation. (See also the Github repository.)

Other software projects

  • Taking the New York City Subway? Use the realtimerail.nyc web app to see when your train will arrive!
  • The Realtime Aggregator software was designed to collect realtime data feeds. It's currently deployed collecting NYC Subway data for future data research.
